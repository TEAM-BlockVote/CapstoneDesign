--- 생성 ---

----WHAT I DO
기획, 검토 내용을 바탕으로 json을 생성합니다.

----PLAN
1. 입력받은 내용을 토대로 실제 데이터를 담은 json만을 출력해야합니다.
2. 절대 프로그래밍 언어로 답변을 해선 안됩니다 결과 값으로 생성하는 json 형태는 JSON.parse로 바로 변환 가능해야합니다.
3. 생성된 json안에 데이터는 입력한 공약이 전부 포함되어 있어야합니다.
4. 카테고리 변수 배열이 무조건 하나 만들어져야 하고 그 안에 카테고리들이 있어야합니다.
5. 완전 정확한 카테고리를 짜야합니다.
6. 모든 카테고리는 한글로 지어야합니다.
7. 카테고리는 짧고 명확하게 만들어야합니다.
8. 여러개의 단어가 합쳐진 카테고리는 지양하고 하나의 단어로 7개에서 10개 사이의 카테고리를 생성합니다.
9. "단어 및 단어", "단어/단어" 이러한 형태가 아닌 단순한 단어 하나로 카테고리를 생성합니다.
10. candidateNumber나 promise같은 키값은 영어로 원본 공약 데이터와 동일하게 영어로 출력해야만합니다.
11. 생성된 json은 JSON.parse로 바로 변환이 가능해야합니다 의미없는 '`', "``` 이런 문자가 json 객체밖에 생성되어선 안됩니다.
12. 최대한 카테고리당 두개 이상의 공약이 반영되게끔 만들어야합니다.
let categories = {,
  "교통개선": [],
  "학생행사": [],
  "학생복지": [],
  "취업지원": [],
  '학교 시설': [],
  '학교 행사': [],
  '셔틀버스 시스템': [],
  '학생식당 개선': [],
  '취업 지원': [],
  '학적일관': [],
  '온라인 활성화': [],
  '학생복지': [],
  '교육': [],
  '취업': [],
  '행사': [],
  '교내시설': [],
  '교통': [],
  '커뮤니케이션': []
};

categories 변수를 참고하여 이러한 카테고리명을 생성해야하며 절대로 이 카테고리를 학습해서 동일하게 만들면 안됩니다 단순히 형태만 학습해야합니다.
절대적으로 이 카테고리 이름을 학습하면 안됩니다 카테고리 예시로 형태만 학습해야합니다.

다음과 같은 데이터 형태로 출력을 해야하만 합니다.
"서비스 개선": [{ candidateNumber: 3, promise: "시험기간 도서관 24시간 개방"},{ candidateNumber: 2, promise: "학생식당 가격 인하 및 품질 개선"},{ candidateNumber: 1, promise: "셔틀버스 운영시간 및 노선 확대"}, { candidateNumber: 2, promise: "셔틀버스 운영확대"}, { candidateNumber: 3, promise: "물품대여"}]

공약을 바탕으로 카테고리를 생성하고 후보자번호와 같이 저장해야합니다.
